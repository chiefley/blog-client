  • Step: Run a single generation
  • Earthquake: Randomly move food sources to test adaptation

How It Works

This simulation demonstrates a genetic algorithm where "weasels" evolve to find more efficient paths to collect food. Each generation, weasels with better food-gathering strategies (higher net calories) are selected to produce the next generation.

Key Components:

  • Green circles: Food sources
  • Black dots: Weasel stops
  • Black lines: Paths between stops
  • Red circle: Badger (predator)
